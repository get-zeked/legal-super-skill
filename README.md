# Legal Super-Skill

Comprehensive legal operations skill merging **Perplexity Computer's 6 legal skills** with **Claude Code's writing, communications, and planning skills**.

## What's Inside

| Section | Description |
|---------|-------------|
| Gap Analysis | Capability comparison matrix |
| Contract Review & Redlining | Clause-by-clause analysis, GREEN/YELLOW/RED classification, redline format |
| NDA Triage | 10-category screening, classification rules, routing |
| Compliance & Privacy | GDPR/CCPA, DPA review, DSR workflows |
| Risk Assessment | Severity×likelihood matrix, escalation criteria |
| Legal Writing & Comms | 7 canned response templates, internal communications |
| Meeting Briefings | Prep methodology, action tracking |
| Verification & QA | Gate functions, per-output checklists |
| Perplexity Capabilities | Live research, 400+ integrations, document synthesis |

## Sources Merged

**Perplexity Computer:** legal-contract-review, legal-nda-triage, legal-compliance, legal-risk-assessment, legal-meeting-briefing, legal-canned-responses

**Claude Code:** Writing Skills, Internal Comms, Writing Plans, Verification Before Completion (obra/superpowers + anthropics/skills)
